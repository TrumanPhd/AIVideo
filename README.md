1) 有些比较偏的场景出现不一致性

3）老模型做real的能力不行 尽量Cartoon风格
4）图片提示时，有些模型会因为擦边图片大尺度而拒绝生成
5）5s 10s可选时 都用5s 10s视频现在的模型都太空洞
